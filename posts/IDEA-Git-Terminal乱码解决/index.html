IDEA Git Terminal 乱码解决

2021.09.07

开发过程中，我习惯将 IDEA 的 Terminal 替换成 Git Bash, 来模拟 Linux 环境的使用，在使用过程中发现，对于中文的显示都是乱码，本文记录下该问题的解决方案。

IDEA Terminal 替换

  • 打开 settings 设置界面，选择 Tools 中的 Terminal (File -> settings -> Tools -> Terminal)
  • 修改 Shell path 为你的 Git bash 安装路径，我的在 D:\Program Files\Git\bin\bash.exe
  • 重新打开你的 Terminal 就能看到变成了经常看到的 Git bash窗口

当你的文件还有中文字符时，会发现回显中文都是乱码，这是由于 Git bash 对中文做了 Unicode 转码造成的。

乱码解决

首先找到 Git 的安装目录中下找到名为的 bash.bashrc 的配置文件，例如我的文件路径为: D:\Program Files\Git\etc， 在该文件的末尾添加如下内容

export LANG="zh_CN.UTF-8"
export LC_ALL="zh_CN.UTF-8"

保存后关闭 IDEA 的 Terminal 面板重新打开，或者重启 IDEA，再次查看中文已经回显正常。
